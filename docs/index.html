From flintlock to LAWS.
250 years of U.S. military tech, mapped as a network.

A network-analysis study of 32 weapon systems and 41 technological relationships (1776–2026), asking how military innovation has shaped trans-Pacific conflict outcomes. Built with Louvain community detection — modularity 0.5362, six communities, four carrying the analytical weight.

Four patterns that drive the thesis

Community 0

Indo-Pacific convergence

Modern U.S., Chinese, and Russian weapons cluster together — Cold-War bipolarity has dissolved into a multipolar techno-strategic field.

Community 1

AI / autonomy rupture

LAWS, Gorgon Stare, Switchblade, Reaper form an isolated paradigm — the first community with no analog-era ancestry.

Community 3

Cold-War learning

Soviet small arms (AK-47, AK-74) integrate into U.S. weapon communities — proof of bidirectional technological transfer, not one-way catch-up.

§4.6.5 overlay

Profit-driven inertia

A military–industrial layer (Lockheed / Raytheon / General Dynamics → F-35, Patriot, JASSM) explains why some legacy platforms survive past their technological relevance.

Reproduce locally

Everything in this repo is reproducible from the CSVs in /data.

git clone https://github.com/szesex/dhga-weaponhistory.git
cd dhga-weaponhistory
pip install pandas networkx python-louvain matplotlib folium plotly
python code/capstone_network_visualization.py
python code/gen_folium_map.py
